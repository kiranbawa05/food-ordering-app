# 🍔 Food Ordering System (Full Stack Mini Project)

A simple full-stack food ordering web application built using **Node.js, Express, and MongoDB**.  
The system allows users to browse food items, add them to cart, place orders, and give feedback. Admins can manage menu items dynamically.

---

## 🚀 Features

### 👤 User Side
- User login system
- Browse food menu
- Search food items
- Add to cart
- Increase / decrease quantity
- Place orders
- View order history
- Submit feedback with star rating

---

### 🧑‍💼 Admin Side
- Admin login system
- Add new food items
- Update existing items automatically (no duplicates)
- Delete food items
- Manage quantity and availability
- Upload item images (via image name/path)

---
